# 自言自语日记

一个轻量的自言自语式日记应用，帮助你记录每日的思考、心情和想法。

## 特点

- **对话式记录** - 以聊天的形式写日记，轻松自然
- **快捷发送** - 双击 Enter 发送，避免误触
- **即时撤回** - 发送后快速撤回，修改重发
- **关键词提取** - 自动从内容中提取关键词
- **分类浏览** - 按关键词快速查找历史日记
- **本地存储** - 所有数据保存在本地浏览器，隐私安全

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

```bash
# macOS
open index.html

# Windows
start index.html

# 或者直接拖入浏览器
```

## 快捷键

- **双击 Enter** - 发送消息
- **撤回按钮** - 点击发送按钮旁边的撤回按钮可撤回上一条

## 数据说明

- 所有日记数据存储在浏览器本地存储 (localStorage) 中
- 清除浏览器数据会导致日记丢失，请注意备份
- 支持导出功能（可在代码中扩展）

## 技术栈

- 纯 HTML/CSS/JavaScript，无任何依赖
- 单文件应用，部署简单
